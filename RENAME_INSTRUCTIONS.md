# GitHub 仓库重命名步骤

## 在 GitHub 网页上重命名仓库

1. **访问您的仓库**
   - 打开：https://github.com/moyinghui/wanyu-li-hui-zhou-project2

2. **进入 Settings**
   - 点击仓库页面右上角的 "Settings" 按钮

3. **重命名仓库**
   - 在 Settings 页面，向下滚动到 "Repository name" 部分
   - 将名称从 `wanyu-li-hui-zhou-project2` 改为 `wanyu-li-hui-zhou-project3`
   - 点击 "Rename" 按钮确认

4. **确认重命名**
   - GitHub 会自动重定向到新名称的仓库

## 重命名后的新链接

**https://github.com/moyinghui/wanyu-li-hui-zhou-project3**

## 注意事项

- 重命名后，旧的链接会自动重定向到新链接
- 所有现有的克隆和 forks 仍然有效
- 如果仓库是公开的，可能需要几分钟让更改生效
